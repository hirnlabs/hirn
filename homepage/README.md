# Hirn Homepage

This is the central web interface and dashboard for the Hirn assistant tooling monorepo. It is built with [Astro](https://astro.build/).

## 🚀 Features

This dashboard provides a unified interface for the various Hirn services:
- **Agent**: AI agent configuration and monitoring.
- **Learn**: Knowledge base and learning resources.
- **Router**: Network and request routing management.
- **Server**: Backend service status and controls.
- **Transcribe**: Speech-to-text processing and management.

## 🧞 Commands

All commands are run from the `homepage/` directory:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `bun install`             | Installs dependencies                            |
| `bun dev`                 | Starts local dev server at `localhost:4321`      |
| `bun build`               | Build your production site to `./dist/`          |
| `bun preview`             | Preview your build locally, before deploying     |
